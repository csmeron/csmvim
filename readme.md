# Introduction
---
This repo is the host/storage for my personal Neovim config.
My config was heavily inspired by both LazyVim and NVChad, as I used to regularly
used NVChad before learning to how make my own config, and then fell in love with
the aesthetic/appearance of LazyVim. Credit goes to those for inspiration. In fact
many settings are the same.

## Features
- Lazy loaded via [lazy.nvim](https://github.com/folke/lazy.nvim)
- Easy and fast theme swapping with [themery.nvim](https://github.com/themery.nvim)
- Easily extendable LSP configuration thanks to [mason.nvim](https://github.com/mason-org/mason)
- And various others.
- More to be added. Currently in a simply functional state

## Prerequisites
- Ensure latest version of Neovim installed, preferably v0.11+.
- Have wget, curl, unzip, make, pip, npm, node, luarocks, ripgrep, cargo, etc. installed.
- Have a nerd font installed.
