# osmnvim

This repo holds my personal Neovim config.

Osmnvim was *very* heavily inspired by both [LazyVim](https://github.com/LazyVim/LazyVim) and [NvChad](https://github.com/NvChad/NvChad), as I diverged from using a "distro" config when wanting to lighten up the load on my weaker systems and trim off the fat I didn't need.
